usage:

  rfileify file

    Inline `rfile()`, `rhtml()` and `ruglify()` calls from `file`, printing the transformed file
    contents to stdout.

  rfileify
  rfileify -

    Inline `rfile()`, `rhtml()` and `ruglify()` calls from stdin, printing the transformed file
    contents to stdout.

N.B.

  You must have `rfile`, `rhtml` and `ruglify` installed to use them:

  npm install rfile rhtml ruglify -g
